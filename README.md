# Менеджер книг на Vue 3

Интерактивное приложение для управления домашней библиотекой.

### Технологии:
- Vue 3 (Composition API)
- Vite
- LocalStorage для сохранения данных

### Функционал:
- Добавление, удаление и поиск книг.
- Система рейтингов (звезды) для прочитанных книг.
- Фильтрация по статусу чтения.
- **Дополнительно:** Возможность помечать книги как "Избранное" (❤️).
